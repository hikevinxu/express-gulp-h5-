# gulp-simple-template
> 简单的 gulp 项目配置，适用于一般简单的xiang项目

- gulp 自动化
- sass 编写样式
- html、css、js 代码压缩，添加MD5
- gulp-autoprefixer 浏览器私有前缀自动补全

## 运行

### 1. 编译并生成开发版本

> `npm run dev` 或者 `gulp dev`

### 2. 本地运行开发版本，支持实时刷新

> `npm run run` 或者 `gulp run`

### 3. 输出生产版本，添加 MD5

> `npm run dist` 或者 `gulp dist`
